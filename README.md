# সত্যের কাফেলা ওয়েলফেয়ার সোসাইটি ফ্যামিলি — সদস্য ডিরেক্টরি

একটি আধুনিক, পরিষ্কার ওয়েবসাইট যেখানে সকল সদস্যদের ব্যবসা, সেবা ও পেশা একসাথে দেখা যায়। সদস্যরা তাদের দোকান খুঁজতে পারেন, যোগাযোগ করতে পারেন এবং সম্পর্কে বিস্তারিত জানতে পারেন।

## 🚀 কীভাবে শুরু করবেন

### ধাপ ১ — Google Sheet তৈরি করুন

একটি নতুন Google Sheet খুলুন এবং **Row 1-এ এই headers লিখুন:**

```
name | category | owner | address | phone | whatsapp | photo | photo2 | description | social | maplink
```

### ধাপ ২ — ডেটা যোগ করুন

প্রতিটি সদস্যের জন্য একটি নতুন সারি তৈরি করুন এবং তথ্য পূরণ করুন।

#### কলাম বিবরণ:

| কলাম | উদ্দেশ্য | উদাহরণ |
|------|---------|--------|
| **name** | দোকান বা প্রতিষ্ঠানের নাম | রহমান মুদিখানা |
| **category** | বিভাগ (যেকোনো নাম লিখতে পারেন) | খাদ্য ও মুদিখানা |
| **owner** | মালিক বা ব্যক্তির নাম | মোঃ আব্দুর রহমান |
| **address** | পূর্ণ ঠিকানা | ৩ নং ওয়ার্ড, সিদ্ধিরগঞ্জ, নারায়ণগঞ্জ |
| **phone** | ফোন নম্বর | +880 1711-000001 |
| **whatsapp** | শুধু সংখ্যা (কোনো +, স্পেস নেই) | 8801711000001 |
| **photo** | প্রথম ছবির imgbb লিংক | https://i.ibb.co/ABC123/shop.jpg |
| **photo2** | দ্বিতীয় ছবির imgbb লিংক (ঐচ্ছিক) | https://i.ibb.co/DEF456/shop2.jpg |
| **description** | ব্যবসার সংক্ষিপ্ত বিবরণ | তাজা শাকসবজি ও মুদিপণ্য |
| **social** | ফেসবুক, ইনস্টাগ্রাম বা ওয়েবসাইট লিংক | https://fb.com/example |
| **maplink** | Google Maps শেয়ার লিংক | https://maps.google.com/?q=... |

> **খালি কলাম রাখতে পারেন** — photo2, social, maplink না থাকলেও কোনো সমস্যা নেই।

## 🖼️ ছবি আপলোড করুন (imgbb ব্যবহার করে)

1. যান → **https://imgbb.com**
2. **"Start uploading"** বাটনে ক্লিক করুন
3. আপনার ছবি সিলেক্ট করুন
4. আপলোড হলে **"Direct Link"** বাটনে ক্লিক করুন
5. লিংকটি কপি করুন — এটা দেখতে এমন হবে: `https://i.ibb.co/ABC123/photo.jpg`
6. এই লিংক Sheet-এর `photo` বা `photo2` কলামে পেস্ট করুন

**গুরুত্বপূর্ণ:** 
- ❌ `https://ibb.co/ABC123` (এটা কাজ করবে না)
- ✅ `https://i.ibb.co/ABC123/photo.jpg` (এটা কাজ করবে)

## 🔗 Google Sheet CSV পাবলিশ করুন

1. আপনার Sheet খুলুন
2. যান → **File → Share → Publish to web**
3. নিচের সেটিংস করুন:
   - **Sheet1** সিলেক্ট করুন
   - **Comma-separated values (.csv)** সিলেক্ট করুন
4. **Publish** বাটনে ক্লিক করুন
5. লিংকটি কপি করুন

লিংকটা এমন দেখতে হবে:
```
https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/pub?output=csv
```

## 📝 ওয়েবসাইটে Sheet যুক্ত করুন

1. `index.html` খুলুন (কোনো এডিটরে — Notepad, VS Code, etc.)
2. এই লাইনটি খুঁজুন:
   ```javascript
   const SHEET_URL = 'YOUR_GOOGLE_SHEET_CSV_URL_HERE';
   ```
3. আপনার CSV লিংক পেস্ট করুন:
   ```javascript
   const SHEET_URL = 'https://docs.google.com/spreadsheets/d/YOUR_ID/pub?output=csv';
   ```
4. ফাইল সেভ করুন

## 🌐 GitHub Pages-এ হোস্ট করুন

### প্রথমবার সেটআপ:

1. **GitHub-এ নতুন repository তৈরি করুন:**
   - `https://github.com/new` যান
   - Repository name: `SKWS-family` (বা যেকোনো নাম)
   - **Public** সিলেক্ট করুন
   - **Create repository** ক্লিক করুন

2. **ফাইল আপলোড করুন:**
   - আপনার রেপোতে যান
   - **Add file → Upload files** ক্লিক করুন
   - এই ৩টি ফাইল আপলোড করুন:
     - `index.html`
     - `logo.jpg`
     - `README.md`
   - **Commit changes** ক্লিক করুন

3. **GitHub Pages সক্রিয় করুন:**
   - রেপো সেটিংসে যান
   - **Pages** (বাম সাইডবার)
   - **Branch:** `main` সিলেক্ট করুন
   - **Folder:** `/ (root)` সিলেক্ট করুন
   - **Save** ক্লিক করুন

4. **লাইভ সাইট দেখুন:**
   - ২-৩ মিনিট অপেক্ষা করুন
   - যান → `https://yourusername.github.io/SKWS-family/`

### আপডেট করতে:

এর পরে যখনই আপডেট করতে চাইবেন:
1. `index.html` এডিট করুন
2. GitHub-এ আপলোড করুন (Upload files → Commit)
3. সাইট স্বয়ংক্রিয়ভাবে আপডেট হয়ে যাবে
