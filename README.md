## 📋 Google Sheet — Column Order

Create your Sheet with **exactly these headers** in Row 1:

| name | category | owner | address | phone | whatsapp | photo | description | social | maplink |
|------|----------|-------|---------|-------|----------|-------|-------------|--------|---------|

### Column guide

| Column | কী লিখবেন | উদাহরণ |
|--------|-----------|--------|
| `name` | দোকান / প্রতিষ্ঠানের নাম | রহমান মুদিখানা |
| `category` | বিভাগ (যেকোনো নাম লিখতে পারেন) | খাদ্য ও মুদিখানা |
| `owner` | মালিক বা ব্যক্তির নাম | মোঃ আব্দুর রহমান |
| `address` | পূর্ণ ঠিকানা | ৩ নং ওয়ার্ড, সিদ্ধিরগঞ্জ |
| `phone` | ফোন নম্বর | +880 1711-000001 |
| `whatsapp` | শুধু সংখ্যা, কোনো স্পেস বা + নেই | 8801711000001 |
| `photo` | imgbb লিংক (নিচে দেখুন) | https://i.ibb.co/... |
| `description` | সংক্ষিপ্ত বিবরণ | তাজা মুদিপণ্য পাওয়া যায় |
| `social` | Facebook / Instagram / Website যেকোনো লিংক | https://fb.com/... |
| `maplink` | Google Maps শেয়ার লিংক | https://maps.google.com/... |

> **খালি রাখলে কোনো সমস্যা নেই।** photo, social, maplink — এগুলো না থাকলে পপআপে ঐ অংশ দেখাবে না।

## 🖼️ ছবি আপলোড করবেন যেভাবে (imgbb)

1. যান → **https://imgbb.com**
2. "Start uploading" বাটনে ক্লিক করুন
3. ছবি সিলেক্ট করুন
4. আপলোড হলে **"BBCode"** বা **"Direct Link"** কপি করুন
5. সেই লিংকটি Sheet-এর `photo` কলামে পেস্ট করুন

> ✅ imgbb ফ্রি, কোনো একাউন্ট ছাড়াও কাজ করে।

## 📂 Category — যেকোনো নাম লিখুন

Category কলামে আপনি **যা খুশি লিখতে পারেন** — সাইট স্বয়ংক্রিয়ভাবে নতুন ফিল্টার বাটন তৈরি করবে। কোনো কোড পরিবর্তন লাগবে না।

**কিছু পরিচিত নাম (আইকোন সহ):**
- `খাদ্য ও মুদিখানা` 🛒
- `পোশাক ও টেইলার` 👗
- `স্বাস্থ্য ও চিকিৎসা` 🏥
- `ইলেকট্রনিক্স ও মেরামত` 🔧
- `শিক্ষা ও প্রতিষ্ঠান` 📚
- `অন্যান্য সেবা` 🤝
- `বিউটি ও সেলুন` 💇
- `পরিবহন` 🚗
- `নির্মাণ ও মিস্ত্রি` 🏗️
- `পেশাদার সেবা` 💼
- `কেটারিং ও হোম ফুড` 🍱

নতুন কোনো নাম লিখলে 🏪 আইকোন দেখাবে — ঠিকঠাক কাজ করবে।

## 🔗 Google Sheet CSV লিংক যোগ করুন

1. Sheet খুলুন → **File → Share → Publish to web**
2. **Sheet1** ও **CSV** সিলেক্ট করুন → Publish → লিংক কপি করুন
3. `index.html` খুলুন, এই লাইনটি খুঁজুন:
   ```
   const SHEET_URL = 'YOUR_GOOGLE_SHEET_CSV_URL_HERE';
   ```
4. লিংকটি পেস্ট করুন।
